## 4. "뼈의 계"

| 항목 | 내용 |
|------|------|
| 유형 | 자연계 |
| 대기 | 부식성 |
| 중력 | 0.6G |
| 온도 | 극한(-60℃) |
| 지형 | 거대한 뼈 구조물 |
| 흑공 | 위험 |
| 자원 | 극대(성물급 뼈) |

땅이 뼈다. 산이 뼈다.
하늘은 검다. 공기는 산성이다.
무언가 거대한 것의 해골 위에
서 있다는 것을 깨닫는 데 1시간.
그것이 아직 살아 있다는 것을
깨닫는 데 1분.

> 시나리오: DCSA가 이 계를
> 영구 봉인하려 한다.
> 봉인 전에 내부 데이터를
> 회수해올 러너가 필요하다.

## 5. "화원"

| 항목 | 내용 |
|------|------|
| 유형 | 자연계 |
| 대기 | 표준 |
| 중력 | 1G |
| 온도 | 온난 |
| 지형 | 끝없는 꽃밭 |
| 흑공 | 무 |
| 자원 | 높음(추모화 원종) |

아름답다. 위험하지 않다.
꽃이 끝없이 피어 있다.
바람이 불면 꽃잎이 날린다.
러너는 이 계에서 돌아오기
싫어진다.

그것이 이 계의 위험이다.
머물고 싶어진다는 것.
1시간마다 의지 판정 DC 12.
실패하면 "조금만 더"라고 생각한다.
3번 실패하면 돌아오지 않는다.

> 시나리오: 이 계에서 돌아오지 않은
> 러너를 데려오라는 의뢰.
> 그 러너는 행복하다.
> 데려가야 하는가?
