## 15-6. 구역 생성 워크드 예시 -- "테이블 3번 굴려서 구역 완성하기"

GM이 즉석에서 구역을 만드는 과정을 보여줍니다.

### 1단계: 구역 생성 (15-1, d6)

결과 4: 서민구/노마드 경계.

> "쪽방이 밀집된 시장 거리. 보따리 장수와 야메가 뒤섞인 곳."

### 2단계: 분위기 결정 (15-4, d10)

결과 3: 야시장.

> "노점, 소음, 연기, 군중. 환경 태그: [군중][연기][노점 장애물]."

### 3단계: 이벤트 (15-5, d10)

결과 6: 정전.

> "갑자기 거리의 돌칩 조명이 꺼진다. 신기 회로 과부하.
> 연기 사이로 누군가 비명을 지른다."

### 완성된 구역

| 항목 | 내용 |
|------|------|
| 구역 | 서민구 경계 야시장 |
| 분위기 | 어둡고 혼잡하고 냄새가 심함 |
| 환경 태그 | [군중] [연기] [노점 장애물] -> 정전 후 [암흑] 추가 |
| 이벤트 | 정전 -- 범죄 기회 또는 러너 의뢰 발생 |
| NPC | 야메(불법 임플란터), 보따리 장수, 노점 주인 |
| 미션 훅 | "정전 틈에 누군가 야메의 금고를 털었다. 야메가 러너를 고용한다. 보수: 오얏 300 + 임플란트 할인" |

이 과정에 1분이면 충분합니다.
테이블 3번 굴리면 구역이, NPC 테이블(15-3)을 추가하면
그 구역의 핵심 인물까지 완성됩니다.
