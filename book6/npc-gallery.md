## Ch.18 주요 인물 (요약)

| 이름 | 설명 |
|------|------|
| 소화 | 힐링카페 마스터. 치코의 파트너. 러너들의 거점. |
| 치코 | 소화의 파트너. 러너. |
| 진 | 신단을 조사하던 신해교수. 어부 볼래와 함께 바다섬에서 정체불명의 괴물을 소환. |
| 크리스 | 메탈 연주자이자 러너. 수많은 사고에 연루된 유명인. 크로코다일을 소유했다는 소문. |
| 한칸 | 괴수왕 야전교수. 흑공 존재와 싸우는 현장의 학자. |
| 레이 | 다리에 장애를 얻은 러너. AI의 대변인 "그림자"가 되어 AI의 인공계를 구축. |
| 에이미 하트 | 최초의 파이오니아. 미모국 개척 전투부대 지휘관. 뇌네 보정 이후 정신 붕괴. "퀸 오브 하트". 하트의 왕국 여왕. |
| 가야 | 신롱물사 가보의 딸. 아버지가 딸을 "걸어다니는 신단"으로 만들었다. 도시급 에너지 공급 가능. |
| 케이브 | 최초의 팝티콘. 배관에 접근하여 입금신호를 변조. 아이들에게 폭죽을 팔았으나 (주)신성에 큰 충격을 준 사건. |
