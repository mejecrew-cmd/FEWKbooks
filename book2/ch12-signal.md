## 12-1. FEWK의 네트워크 — 전자가 없는 세계

검은 수평선 이후 전자(Electronic)는 작동을 멈췄습니다.
FEWK에는 인터넷도, 컴퓨터도, 와이파이도 없습니다.

대신 있는 것:
- 흑공 기반 계산기(Computing Device) — 기억력/연산력/판단력 3축.
- 신기 회로 네트워크 — 신단석을 연결한 물리적 회선.
- 심상 전달 — 이미지를 직접 정신으로 전달하는 통신.
- 개인통신채널 — 임플란트 기반 단거리 통신.

해킹은 사이버스페이스가 아니라, 신기 회로에 물리적으로
접근하여 돌칩과 신단석의 배선을 조작하는 행위입니다.

## 12-2. 신호연결 체계 Signal Connection System

신호연결사(神号連結士)의 핵심 활동입니다.

### 접근 방식

"물리적 접근 필요"를 그대로 계승합니다.
신호연결사는 팀과 함께 현장에 있어야 합니다.
원격 해킹은 극히 제한적(심상전이 능력자만 가능).

### 노드(Node) 구조

보안 시설의 신기 회로는 노드 구조로 설계됩니다.
네트워크 아키텍처를 FEWK식으로 번안:

- 입구 노드(Entry Node) — 외부 접근점. 물리적 접촉 필요. 돌칩 커넥터나 노출된 신단석 배선을 찾아야 함.
- 보안 노드(Security Node) — 방화벽. 돌파하지 않으면 진행 불가.
- 기능 노드(Function Node) — 문 열기, 카메라 제어, 경보 등.
- 핵심 노드(Core Node) — 데이터 저장소. 목표 정보 소재지.
- 수호 노드(Guardian Node) — 능동 방어. 침입자를 공격.

### 해킹 판정

> 해킹 체크 = 2d6 + 혜(INT) 수정치 + 신호(Signal) 스킬
> 또는 Program 스킬 (흑공 계산기 조작 시)
>
> 보안 노드 난이도: 약(8) / 중(10) / 강(12) / 기업급(14)
> 실패 시: 경보 발동 또는 수호 노드 활성화.

### 수호 노드의 공격

수호 노드가 활성화되면 신호연결사에게 역공격합니다.

| 등급 | 피해 | 추가 효과 |
|------|------|----------|
| 약 | 1d6 | 시스템 부하에 적용 |
| 중 | 2d6 | 돌칩 장비 과부하 위험 |
| 강 | 3d6 | 접촉 장비 파괴 가능 |
| 기업급 | 4d6 | 정신 내성 체크 (실패 시 기절) |

피해는 시스템 부하(System Strain)에 먼저 적용됩니다.
시스템 부하 최대 초과 시: 의식 상실 + 접촉 중인 임플란트 손상.

## 12-3. 프로그램 — 돌칩 소프트웨어

Programs를 FEWK에서는 "돌칩 프로그램"이라 부릅니다.
신호연결사가 해킹 시 장비하여 사용하는 도구.

### 공격형 프로그램

- 회로단절(Circuit Break) — 보안 노드 1개 비활성화. 1라운드 지속.
- 과부하(Overload) — 기능 노드 1개에 신기 과부하. 영구 파괴. 대상 시설에 물리적 피해 (문 폭발, 조명 파괴 등).
- 신폭 유도(Sinki Trigger) — 신기 회로에 의도적 신폭 유발. 반경 3m 피해 2d6. 해커 본인도 위험.
- 역추적 공격(Counter-Trace) — 자신을 추적 중인 해커에게 역공격. 대상 시스템 부하 +2. 추적 시계 -1.
- 잠금 해제(Lock Crack) — 잠긴 문/금고의 기능 노드 강제 개방. 난이도 = 노드 난이도 +2. 성공 시 영구 해제.

### 방어형 프로그램

- 차폐(Shield) — 수호 노드 피해 절반 감소. 3라운드 지속.
- 은폐(Cloak) — 경보 발동 1라운드 지연. 실패 시 효과 없음.
- 우회(Bypass) — 보안 노드 1개를 우회. 난이도 -2 효과.
- 흔적 소거(Wipe) — 해킹 흔적 삭제. 추적 시계 -2. 1회성.
- 미끼 신호(Decoy) — 가짜 위치 신호 생성. 추적자 혼란 1d4 라운드.

### 유틸리티 프로그램

- 탐색(Scan) — 연결된 노드 구조 전체 파악. 1라운드 소모.
- 복제(Copy) — 핵심 노드의 데이터를 돌칩에 복사. 크기에 따라 시간.
- 삽입(Insert) — 가짜 데이터 삽입. Talk 체크 필요(속이는 데이터 품질).
- 추적(Trace) — 통신 경로 역추적. 발신지 특정.
- 도청(Wiretap) — 통신채널 감청. 10분 지속. Notice 체크로 탐지 가능.

프로그램은 돌칩에 저장되며, 한 번에 장비 가능한 수 = 혜(INT) 수정치 + 2.

## 12-4. 특수 해킹 — 임플란트 재밍 & 심상 침투

### 임플란트 재밍(Implant Jamming)

근거리(10m 이내) 무선으로 적의 임플란트를 일시 방해.
- 체크: 2d6 + 혜(INT) + Program >= 대상의 SC% / 10 + 8
- 성공 시: 대상 임플란트 1개를 1라운드 비활성화.
- 사이버아이 재밍: 시각 상실 1라운드. 공격 -4.
- 통신채널 재밍: 통신 두절 1라운드.
- 한계: 1라운드만 지속. 대상이 정신 내성 성공 시 즉시 해제.

### 심상 침투(Image Intrusion)

심상(Image) 스킬 + 심상전이 능력 보유자만 가능.
대상의 정신에 심상을 강제 전송하는 공격.
- 체크: 2d6 + 감(SEN) + 심상(Image) vs 대상 정신 내성
- 성공 시: 환각, 혼란, 거짓 정보 주입 (1라운드).
- 대실패(자연 2): 역류 — 시전자가 대상의 심상에 노출. 매팅 트랙 +1 위험.

## 12-5. 노드 유형과 해킹 난이도 Node Types & Difficulty

모든 신기 네트워크는 노드(Node)로 구성됩니다.
해킹 대상의 노드 유형에 따라 난이도와 위험이 다릅니다.

| 노드 유형 | 난이도 | 특징 |
|----------|--------|------|
| 공개 노드 | 6 | 공공 정보. 거리 조명, 광고판. 실패해도 추적 없음. |
| 보안 노드 | 8 | 건물 보안, CCTV, 도어락. 실패 시 경보. 증원 시계 +1. |
| 기업 노드 | 10 | 기업 내부망. 인사/재무/생산 데이터. 실패 시 추적 시계 활성(4칸). |
| 군사 노드 | 12 | 대사국/수호사 보안망. 실패 시 즉시 추적+역공격(1d6 에고). |
| 신단 노드 | 14 | 신단 제어 회로. 극비. 실패 시 신폭 위험. 신폭 시계 +2. |

- 해킹 체크: 2d6 + 혜(INT) + Program(또는 신호) >= 난이도.
- 1라운드 1노드. 복수 노드 해킹은 연속 체크 필요.
- 보안 프로그램: 기업/군사 노드에는 방어 프로그램(ICE)이 있음. ICE 돌파: 추가 체크(난이도 = 노드 난이도). 실패 시 시스템 부하 +1.

## 12-6. 해킹과 전투 시스템 연동 Hacking & Combat Integration

전투 중 해킹 행위는 전투 규칙과 동일한 프레임워크로 작동합니다.

### 환경 태그 연동

- 해킹으로 환경을 변경하면 새 환경 태그가 생성됩니다.
  예: 조명 해킹 → [암흑] 태그. 보안 카메라 해킹 → [감시 해제] 태그.
- 태그 활성화: 혜(INT) + Program 또는 신호(Signal) 체크(난이도 10).

### 시계 연동

- 해킹 시도가 보안 노드에 탐지될 때마다 → 증원 시계 +1.
- 기업 시스템 해킹 시 → 추적 시계(4칸) 자동 활성. 4칸 차면: 해커 위치 특정. 흑전사 출동.

### 푸시 연동

- 해킹 체크(2d6) 실패 시 푸시 가능. 대가: 시스템 부하 +1 (임플란트 과열) 또는 추적 시계 +1.
- "성공했지만 흔적을 남겼다."
